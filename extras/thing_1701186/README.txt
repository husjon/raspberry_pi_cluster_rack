# Summary

3D Models of the latest Raspberry Pi 3 Model B. Can be used to test print many things for the Raspberry Pi.

Design is based on documentation from the Raspberry Pi Foundation.
All the major components are present as well as all the smaller resistors, capacitor, chips, etc. 

Original files (Solidworks, STEP, IGES) on [GrabCAD](https://grabcad.com/library/raspberry-pi-3-reference-design-model-b-rpi-raspberrypi-raspberry-pi-2)

UPDATE: Added a Light version of the assembly, for easier importing into 3D Printing Software.
